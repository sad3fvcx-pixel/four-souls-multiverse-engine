# Four Souls Multiverse Engine
# Target Registry

Version: 1.0 (Draft)

---

# Назначение

Target определяет объект,
над которым выполняется Effect.

Каждый эффект может использовать один или несколько Target.

Target никогда не изменяет состояние игры.
Он лишь возвращает набор объектов.
