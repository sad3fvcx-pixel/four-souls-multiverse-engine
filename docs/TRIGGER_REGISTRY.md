# Four Souls Multiverse Engine
# Trigger Registry

Version: 1.0 (Draft)

---

# Назначение

Trigger — событие, которое начинает выполнение способности.

Каждая способность карты начинается именно с Trigger.

Один Trigger может запускать любое количество Effect.

Карты никогда не опрашиваются постоянно.
Движок лишь рассылает события.
