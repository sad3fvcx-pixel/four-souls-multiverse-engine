# docs/EFFECT_REGISTRY.md

# Four Souls Multiverse Engine
# Effect Registry

Version: 1.0 (Draft)

---

# Назначение

Данный документ описывает все встроенные эффекты FSME.

Каждый эффект является атомарной операцией.

Карты никогда не содержат игровую логику.
Они лишь комбинируют встроенные эффекты.

---

# Формат описания

Каждый эффект содержит

- Identifier
- Description
- Parameters
- Target
- Stack behaviour
- Return value
